## インストール

### クローンする

- このレポジトリをクローンする

```shell
$ git clone https://github.com/mizhstreet/mimin3.git
```

### セットアップ

- 最初に expo-cli をインストール:

> npm で

```shell
$ npm install -g expo-cli
```

> yarn で

```shell
$ yarn global add expo-cli
```

- セットアップ:

> npm で

```shell
$ npm install
```

> yarn で

```shell
$ yarn install
```

---

## 使い方

```shell
$ yarn start
```
